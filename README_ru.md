# Once Human Tools

Once Human Tools сейчас состоит из трёх поддерживаемых сборок:

- Основная программа `V4`
- Портативный бот для рыбалки `V4`
- Портативный калькулятор `V2`

## Что умеет проект

- Подмена фото и видео в игровых папках
- Бот для пианино с загрузкой и проигрыванием MIDI
- Бот для рыбалки с проверкой OCR, автоматической установкой OCR, Telegram-уведомлениями и работой по зонам
- Калькулятор урона с оружием, бронёй, сетами, модами и игровыми иконками
- Набор инструментов для вытаскивания данных из игры и синхронизации базы калькулятора

## Структура проекта

- Точка входа основной программы: `main.py`
- Портативная рыбалка: `onli_fish_bot/main.py`
- Портативный калькулятор: `portable_calc_v2/main.py`
- Инструменты для данных калькулятора: [`tools/README_ru.md`](tools/README_ru.md)
- Политика использования: [`README_policy_ru.md`](README_policy_ru.md)

## Запуск

Установка зависимостей:

```powershell
pip install -r requirements.txt
```

Запуск основной программы:

```powershell
python main.py
```

Запуск портативной рыбалки:

```powershell
python onli_fish_bot/main.py
```

Запуск портативного калькулятора:

```powershell
python portable_calc_v2/main.py
```

## По рыбалке

- Перед стартом отслеживания сначала проверь OCR в интерфейсе.
- Если Tesseract или языки `eng` / `rus` не установлены, используй встроенную автоустановку.
- Создание зон теперь запускается сразу через системный Windows snipping, а после захвата открывает редактор выделенных зон.

## По калькулятору

- База калькулятора уже расширена оружием, бронёй, сетами, модами и игровыми иконками.
- Параметры персонажа теперь показываются читаемыми названиями, а не сырыми ключами вроде `crit_rate_percent`.
- HP манекена в интерфейсе отображается округлённо.
